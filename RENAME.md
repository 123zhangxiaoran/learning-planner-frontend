# 项目信息

## 项目名称
`learning-planner-frontend`

## 项目简介

这是一个基于 Vue 3 + Vite + TypeScript 的学习规划平台前端项目，主要功能包括：

- **职业探索**: AI 辅助职业规划与探索
- **技能分析**: 个人技能评估与分析
- **知识库存**: 学习笔记与知识管理
- **个人中心**: 用户个人信息管理
- **AI 对话**: 智能助手交互功能

## 技术栈

- Vue 3 + Vue Router
- TypeScript
- Vite
- Element Plus
- Pinia (状态管理)
- ESLint + Prettier
